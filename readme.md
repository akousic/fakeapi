# Fake API 
#### Rest api to simulate api calls for testing

### To run this app
1. Make sure you have node installed
2. Clone project
3. Run ```npm install```
4. Run ```node index.js```


### Invoke API 

1. Health Endpoint
   ```http://localhost:{port}/health```

2. FD/RD
   ```http://localhost:{port}/api/fd```

3. EMI
   ```http://localhost:{port}/api/emi```

4. Card
   ```http://localhost:{port}/api/card```
